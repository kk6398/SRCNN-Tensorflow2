# SRCNN-Tensorflow2
SRCNN-Tensorflow2 for CIFAR10 Dataset

# 说明

该部分代码实现了CIFAR10数据集扩展，实现超分辨率(128,128,3)。

![orig](https://zhaoguanghe.github.io/imgs/nearest.png)<br>

文件说明：

- automobile.png: CIFAR10测试图像

- ```Python3```
- ```TensorFlow 2.0```
- ```numpy```
